### Dataset

The data is retreived from the MillionSong Dataset and pre-processed in R.

#### Features

The features present are the following; 

- TrackID
- ArtistID
- SongTitle 
- Loudness
- Tempo 
- Mode
- ModeConf
- Key
- KeyConf
- Valence 
- Energy (Arousal)
